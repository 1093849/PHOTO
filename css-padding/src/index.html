Padding 內距

外距

padding 屬性可以控制區域如 DIV 或 span 的內部距離，也可以控制 HTML 的表格內部距離（例如文字或圖片與邊框的距離），俗稱內距

　
CSS內距

外距設定

  四邊內距相同→padding:內距;
  四邊外距相同→padding:內距;
  上下、左右邊內距相同→padding:上下邊內距 左右邊內距;
  上、左右、下分開設定→padding:上邊內距 左右邊內距 下邊內距;
  四邊內距分開設定→

  • padding:上邊內距 右邊內距 下邊內距 左邊內距;
  • padding-top:上邊內距;
    padding-right:右邊內距;
    padding-bottom:下邊內距;
    padding-left:左邊內距;
