Margin 外距

外距

margin 屬性用來定義一個區域（例如 DIV 或 span）的外邊界距離，俗稱外距，可設定負值

CSS外距 　

外距設定

  四邊外距相同→margin:外距;
  四邊外距相同→margin:外距;
  上下、左右邊外距相同→margin:上下邊外距 左右邊外距;
  上、左右、下分開設定→margin:上邊外距 左右邊外距 下邊外距;
  四邊外距分開設定→

  • margin:上邊外距 右邊外距 下邊外距 左邊外距;
  • margin-top:上邊外距;
    margin-right:右邊外距;
    margin-bottom:下邊外距;
    margin-left:左邊外距;
