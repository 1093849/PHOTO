Gradients 漸層效果

線性漸層

  線性漸層的方向可以是由上而下漸層、由下而上漸層、由左而右或由右而左漸層，甚至可以設定漸層角度
  background: linear-gradient(方向,顏色1,顏色2, ......);



徑向漸層

  徑向漸層的部份則可以使用預設的橢圓形（ellipse）或自定為圓形（circle）漸層效果
  background: radial-gradient(ellipse / circle,顏色1,顏色2, ......);
