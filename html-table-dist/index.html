<table 樣式設計> </table>➝表格
<tr 樣式設計> </tr>➝一(橫)行
<td 樣式設計> </td>➝一(直)列
  樣式設計➝width：控制 table 的寬度
       border：控制 table 邊框的粗細
       background：控制背景圖片
<thead> </thead>➝表格表頭區
<tbody> </tbody>➝表格內容區
<tfoot> </tfoot>➝表格頁腳區
<caption> </caption>➝表格大標(一個表一個)
<td rowspan="數字">➝垂直合併儲存格，數字填的就是合併幾個橫列
<td colspan="數字">➝水平合併儲存格，數字填的就是合併幾個直行
<colgroup></colgroup>➝表格直行分組
